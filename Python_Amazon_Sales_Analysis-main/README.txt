# Amazon Sales Report Project

This project is designed to analyze and generate sales reports for Amazon orders. The purpose of the report is to track sales data such as order IDs, product names, quantities, prices, and order dates.

## Features

- *Sales Data Extraction:* Pulls order data from Amazon using a set of APIs or file downloads.
- *Custom Filters:* Allows users to filter reports based on date ranges and product categories.
- *Export Options:* Provides the option to export the generated report in CSV or Excel format.
- *Data Visualizations:* Displays key insights into the sales data, including total sales, most sold products, and sales trends.
